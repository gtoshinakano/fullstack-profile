## Observations (no regression weight — originally 🟡)

- The Gemini API key (`NEXT_PUBLIC_GEMINI_API_KEY`) is client-side. A future re-extraction should note whether HTTP referrer restriction was configured in Google AI Studio.
- `@google/generative-ai` was installed at `^0.24.1`. Future re-extractions may detect a version bump.
- The `event()` helper type (`IEvent.params`) was constrained to `event_category` and `event_label` only. The `ai_terminal_query` GA event uses `event_label` to encode `questions_remaining` as a string — this is a workaround documented in `progress.jsonl#T010`.
