## High-Risk Change Scenarios

### Scenario A: Add a new locale (e.g., `zh`)
**Files to change:**
1. `next-i18next.config.js` — add `'zh'` to `locales[]`
2. `public/locales/zh/common.json` — create translation file
3. `ChangeLanguage/index.tsx` — add language option
4. `next.config.js` — if using custom redirect patterns
5. `src/lib/languageDetector.js` — update fallback mapping

**Risk:** If `getStaticPaths` is not updated, the new locale page is not built.

---

### Scenario B: Add a new project to `toshi-projects.json`
**Files to change:**
1. `src/data/toshi-projects.json` — add entry
2. `public/locales/{en,ja,pt-BR}/common.json` — add i18n keys for title/description
3. **⚠️ Use a unique `label`** — duplicate `label: "requirement"` bug is already present; adding another duplicate worsens the accordion behavior

**Risk:** Low — isolated to Projects component.

---

### Scenario C: Change GitHub Pages repository name (basePath)
**Files to change:**
1. `NEXT_PUBLIC_BASE_PATH` env var in GitHub Actions
2. `next.config.js` `assetPrefix`
3. `next.config.js` `basePath` (if set explicitly)

**Risk:** If any of these is missed, the site loads a blank page or redirects to the wrong path.

---

### Scenario D: Remove GSAP
**Impact on:**
- `HeroSection/WideScreen.tsx` — parallax entirely GSAP-driven; full rewrite required
- `HeroDark/index.tsx` — tab transition animations; photo carousel ScrollTrigger
- `gabriel.tsx` — Pace.js integration is independent; unaffected

**Risk:** 🔴 High — GSAP is deeply coupled to the hero and profile animations. Not a casual dependency swap.

---

### Scenario E: Migrate to Next.js App Router
**Impact on:**
- `pages/` directory — entire Pages Router structure replaced
- `next-i18next` — not compatible with App Router; requires migration to `next-intl` or similar
- `getStaticPaths` / `getStaticProps` — replaced by `generateStaticParams`
- `pages/_app.tsx` — replaced by `app/layout.tsx`
- `src/lib/getStatic.js` — obsolete

**Risk:** 🔴 High — major migration; touches every page and i18n integration.
