- [ ] T-03 — Criar `src/lib/redirect.js` com hook `useRedirect`
  - Origem no legado: `src/lib/redirect.js`
  - **Decisão do autor (Pergunta 7):** o `useEffect` sem deps foi intencional (detectar mudanças de rota). Alternativa recomendada: usar `[router.asPath]` como array de dependências 🟢

    - Preserva o comportamento original: roda na primeira hidratação e sempre que o path muda
    - Evita a execução redundante em re-renders não relacionados a mudança de rota
    - O guard `router.asPath !== target` permanece como segurança adicional
  - Critério de pronto: `useEffect` com `[router.asPath]`; guard `pathname === '/404'`; detecta locale via `languageDetector.detect()`; compõe target com `prefix + '/' + locale + '/dev/gabriel-toshinori-nakano/'`; executa `router.push(target)` apenas se `router.asPath !== target`
  - Confiança: 🟢
