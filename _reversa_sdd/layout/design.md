## Interface

| Símbolo | Assinatura | Retorno | Observação |
|---------|-----------|---------|------------|
| `Public` | `({ children, title?, description? })` | `JSX.Element` | Layout shell; envolve todas as páginas |
| `ChangeLanguage` | `()` | `JSX.Element` | Widget autônomo; sem props |
| `prefix` | `() => string` | `string` | Helper — retorna basePath |
| `customLoader` | `({ src }: { src: string }) => string` | `string` | Next.js image loader identity function |

**`prefix.ts`:**
```ts
const prefix = process.env.NEXT_PUBLIC_BASE_PATH ?? ''
export default prefix
// produção: '/fullstack-profile'
// dev:      ''
```

**`customLoader.ts`:**
```ts
const customLoader = ({ src }: { src: string }) => src
export default customLoader
// Retorna src inalterado — bypass completo do optimizer Next.js
```

## Fluxo Principal — Montagem do Shell

1. `pages/_app.tsx` envolve toda a aplicação com `appWithTranslation` e renderiza `<Component {...pageProps} />`
2. Cada página (ex: `gabriel.tsx`) renderiza `<Public title="..." description="...">` como wrapper
3. `Public` injeta no `<Head>` do Next.js:
   - `<title>` e `<meta name="description">`
   - `<meta>` de viewport e OG tags
   - `<link>` para Google Fonts (Syne, variantes 400/500/600/700/800)
   - `<link>` para Unicons CSS CDN
   - `<script>` para Pace.js CDN (blocking — deve estar no head para capturar recursos)
4. `Public` renderiza `<ChangeLanguage />` no layout da página (header ou overlay)
5. `Public` renderiza `{children}` — o conteúdo da página

## Fluxo Principal — Resolução de Asset URL

```
Componente precisa referenciar /img/hero-layer-01.png

  → import prefix from 'src/helpers/prefix'
  → src={prefix + '/img/hero-layer-01.png'}

desenvolvimento:   '' + '/img/hero-layer-01.png'   = '/img/hero-layer-01.png'
produção:  '/fullstack-profile' + '/img/hero-layer-01.png' = '/fullstack-profile/img/hero-layer-01.png'
```

## Fluxo Principal — ChangeLanguage

1. Widget lê locale atual via `useRouter().locale` ou `useRouter().asPath`
2. Renderiza 3 botões/links: EN / 日本語 / PT-BR
3. Locale atual destacado visualmente (ex: bold, cor diferente)
4. Ao clicar em outro locale: `router.push(prefix + '/' + locale + '/dev/gabriel-toshinori-nakano/')`
5. Next.js navega client-side; `appWithTranslation` atualiza o contexto de i18n

## Fluxos Alternativos

- **`NEXT_PUBLIC_BASE_PATH` não definido:** `prefix.ts` retorna `''` (nullish coalescing) — comportamento de desenvolvimento; assets carregam corretamente em localhost mas quebram em produção
- **CDN do Pace.js indisponível:** `<script>` falha em carregar → `window.Pace` é `undefined` → `Pace.on('done', cb)` em `gabriel.tsx` lança TypeError → portfólio fica oculto indefinidamente. Sem fallback.
- **CDN do Unicons indisponível:** `<link>` falha → icon font não carrega → ícones mostram caracteres de fallback (texto vazio ou unicode box) — degradação graciosa, funcionalidade não impactada

## Dependências

- **`next/head`** — `<Head>` component para injeção de metadados
- **`next/router`** — `useRouter()` no `ChangeLanguage` para rota atual e `router.push`
- **`prefix.ts`** — auto-dependência: usado internamente e exportado para outras features
- **`NEXT_PUBLIC_BASE_PATH`** — variável de ambiente pública; deve ser `/fullstack-profile` em produção
- **Google Fonts CDN** — Syne typeface
- **Pace.js CDN** — carregamento do indicador de progresso
- **Unicons CDN** — icon font

## Estado Interno

Nenhum. `Public` e `ChangeLanguage` são stateless (exceto o estado de rota via `useRouter`). `prefix.ts` e `customLoader.ts` são funções puras.

## Observabilidade

Nenhum evento GA específico desta unit. O carregamento de scripts CDN não é monitorado ativamente — falhas de CDN são silenciosas (sem alertas ou logs).

## Riscos e Lacunas

- 🔴 **Pace.js CDN sem fallback:** se o script não carregar, o portfólio inteiro fica oculto. Mitigação: adicionar timeout em `gabriel.tsx` que força `loading=false` após N segundos.
- 🟡 **Pace.js sem pinagem de versão:** a URL CDN sem versão pode receber breaking changes. Considerar hospedar o script localmente em `public/`.
- 🟡 **Sem Content Security Policy (CSP):** scripts e estilos de CDN externos sem hash ou nonce. Não é problema para um portfólio estático público, mas vale documentar.
- 🟡 **`NEXT_PUBLIC_BASE_PATH` não está em `.env.example`:** novos contribuidores podem não saber configurar a variável corretamente.
